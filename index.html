GCRBA Web Application

My roles for this project include writing C# using the ASP.NET MVC framework, as well as SQL Server, JavaScript, HTML, CSS, and Bootstrap to create the following:

  • Log in and log out functionality that also tracks if the user is a member, non-member, or admin
  • Functionality for members and non-members to log in and view/update account information + request changes to be made regarding their bakery/store
  • Functionality for members and non-members to view messages and mark them as read or unread, and to delete them
  • Functionality for members to update specials, events, location information, and what they offer by location
  • Functionality for members to update contact information by location or for all locations
  • Functionality for the admin to view requests + approve or deny them
  • Functionality for the admin to edit bakery information
  • Functionality for the admin to add new bakeries
